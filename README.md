# Real-Time Chat Application

A full-stack web application built with Node.js, Express, and Socket.io.

### Key Features:
- Real-time messaging using WebSockets.
- User authentication with password hashing (bcrypt).
- Relational database integration (MySQL).
- Admin features: clear chat functionality.

### Tech Stack:
- Backend: Node.js, Express
- Real-time: Socket.io
- Database: MySQL
- Frontend: EJS, CSS, Vanilla JS
