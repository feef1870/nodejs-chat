# Real-Time Chat Application

A full-stack web application built with **Node.js, Express, and Socket.io**.

### Key Features:
- [cite_start]Real-time messaging using WebSockets[cite: 7].
- [cite_start]User authentication with password hashing (bcrypt)[cite: 2].
- [cite_start]Relational database integration (MySQL)[cite: 2].
- [cite_start]Admin features: clear chat functionality[cite: 3].

### Tech Stack:
- Backend: Node.js, Express
- Real-time: Socket.io
- Database: MySQL
- Frontend: EJS, CSS, Vanilla JS
